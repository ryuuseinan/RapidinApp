# RapidinApp

# Instalación del repositorio
```
git clone https://github.com/ryuuseinan/RapidinApp
```

# Instalación de dependencias
```
npx expo install @expo/webpack-config
```

# Ejecutar la app
```
cd RapidinApp
expo start
```
